🌈 Color Vocab Website
A simple and visually engaging website that helps users learn and explore color vocabulary in English. Built using only HTML and CSS, this project showcases a clean, responsive layout with colorful sections representing various hues, shades, and their corresponding names.

🔧 Features
Pure HTML and CSS — no JavaScript

Colorful cards with names and HEX codes

Fully responsive design

Easy to customize and extend

📁 Use Cases
Educational tool for kids or language learners

Design inspiration for web color palettes

Lightweight project for practicing front-end skills
